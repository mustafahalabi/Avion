# Company — Engineering OS

**Status:** Approved
**Version:** 1.0
**Owner:** CTO
**Approved By:** CEO
**Last Updated:** 2026-06-29

This directory holds the foundational organizational documents that define the virtual software company at the heart of Engineering OS — how it operates, how it executes work, and how its employees are specified. These documents describe **company behavior**, not software architecture, AI prompts, or implementation technology. Everything in the product (UI, workflows, runtime) derives from them.
